## 注意事项

1. **建议 DHCP 保留**：本机已固定静态 IP，但路由器/DHCP 服务器若不知此地址被占用，极端情况下可能把 `172.16.120.79` 分给其他机器造成冲突。建议在 DHCP 服务器上做 MAC-IP 绑定或地址排除。
2. **网关禁 ping**：网关 `172.16.120.1` 不响应 ICMP，ping 失败属正常，不代表断网（已用 TCP 53 端口实测外网连通确认）。
3. **依赖环境**：若 Python 环境被重置，需 `python -m pip install -r requirements.txt` 重新装全依赖（flask/flask-compress/flask-cors/flask-sqlalchemy/werkzeug/openpyxl/requests）。
4. **重启后服务不自动起**：确认 `SHICHETAI-4` 启动文件夹下的 `tooling-autostart.lnk` 存在；手动验证可执行 `D:\tooling-management\backend\start_flask.bat`。服务崩溃后不会自动重启（仅开机/登录时拉起一次），需人工重启或后续改造为 Windows 服务。
